# RZR Messenger - Official Releases 🚀

Welcome to the official distribution repository for **RZR Messenger**. 

This repository is exclusively used for hosting the compiled `.apk` release files and version updates for the application. Our in-app OTA (Over-The-Air) update system directly fetches the latest app versions from this repository.

> **🔒 Security Note:** > The source code for RZR Messenger is proprietary and hosted in a separate private repository to ensure top-level security and privacy. No source code is exposed here.

## 📥 How to Download
To manually get the latest version of RZR Messenger:
1. Go to the [Releases](https://github.com/YOUR_GITHUB_USERNAME/rzr_releases/releases) section on the right side of this page.
2. Click on the most recent version (e.g., `v1.0.2`).
3. Download the `.apk` file under the **Assets** dropdown and install it on your Android device.
